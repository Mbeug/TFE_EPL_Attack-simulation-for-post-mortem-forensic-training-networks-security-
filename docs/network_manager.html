gns3_request_get(endpoint)¶

Request GET on the gns3 server.

Parameters

endpoint (str) – the endpoint’s path where the request GET is to be made

Returns

Response from the requests class of the GET request at the endpoint

Return type

requests.models.Response

Example:

>>> gns3_request_get('/gns3vm/engines')
>>> gns3_request_get('path/unknown')
# when the path is unknown by the server the response is 400

This will output:

<Response [200]>
<Response [400]>
