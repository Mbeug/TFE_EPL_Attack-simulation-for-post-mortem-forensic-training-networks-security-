analyser_manager module¶

class analyser_manager.AnalyserManager(networkManager: network_manager.NetworkManager)¶

Bases: object

This class is used to manage the analysis of the network

get_all_link()¶

This method allows you to retrieve all the links of the topology.

Returns

A list of all links

start(node)¶

This method call the start_node() method in network_manager. It start a specific node of our topology.

Parameters

node – The specific node

start_all_capture()¶

This method apply on all links of our network the start_capture_link() method of the network_manager. It start the capture on all links.

Note

The pcap file is stored in the capture directory of the GNS3 project

start_capture_btw(nodeA, nodeB)¶

This method start a capture with the start_capture_link() method of the network_manager

Parameters

  • nodeA – One of the nodes

  • nodeB – One of the nodes

Note

The pcap file is stored in the capture directory of the GNS3 project

Warning

nodeA and nodeB must be next to each other
