attacker_manager module¶

class attacker_manager.AttackerManager(topology_manager: topology_manager.TopologyManager)¶

Bases: object

Class to manage attackers in the network. This class calls some methods of the docker_manager class

attacker_config()¶

This method configure attackers pcs

create_attackers(nb_attackers, flag_position)¶

Method creating nb_attackers at the flag_position

Parameters

  • nb_attackers – The number of attackers we want to create

  • flag_position – The position( ‘out’ or ‘in_service’ or ‘in_lan’) where we want to put attackers

Returns

Add to the list of attackers the new attackers

dns_tunneling()¶

This method launch a dns tunneling

dos(attacker_pc, ip_src, ip_dst, port)¶

This method launch a Dos attack, it represents the execution of what a bot in a botnet could do.

Parameters

  • attacker_pc – The bot

  • ip_src – The ip bot

  • ip_dst – The target of the attack

  • port – The port number

Returns

Write in the out_dos_attack.txt the result of the execution

static get_attackers(list_pcs)¶

This method allows you to have the list of attackers.

Parameters

list_pcs – The list of pcs present in the topology

Returns

the list of all attackers present in the topology

host_discovery(attacker_pc, ip_dst, start_range, end_range)¶

This method launch the host discovery with some parameters

Parameters

  • attacker_pc – which attacker launch this attack

  • ip_dst – The mask of the ip address

  • start_range – The number of the port on which the discovery is to begin

  • end_range – The number of the port on which the discovery is to be completed

Returns

Write in the out_discovery_host.txt the result of the execution

Note

The file is stored in out directory

scan_port(attacker_pc, ip_dst, start_port, end_port)¶

This method launch a scan of ports on the pc at a specific ip address with the pc attacker.

Parameters

  • attacker_pc – The pc attacker

  • ip_dst – The target of the scan

  • start_port – The starting port number

  • end_port – The ending port number

Returns

Write in the out_scan_ports.txt the result of the execution
